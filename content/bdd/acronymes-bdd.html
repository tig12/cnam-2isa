Acronymes BDD

SigleSignificationCommentaires
DBA Data Base Administrator Administrateur d'une Base de données, a la charge des GRANT et REVOKE.
DF Dépendance Fonctionnelle
CRUD CREATE READ UPDATE DELETE En sql : CREATE SELECT UPDATE DELETE
DBMS Data Base Management System = SGBD
FK Foreign Key Clé étrangère
FN Forme Normale Traduction de "Normal Form" (voir NF)
NF1 = forme normale 1 etc.
LCD Langage de Contrôle de Données SQL : GRANT, REVOKE
LDD Langage de Définition de Données SQL : CREATE, ALTER, DROP
LID Langage d'Interrogation de Données SQL : SELECT, DESCRIBE (= DESC)
Parfois classsé avec LMD.
LMD Langage de Manipulation de données SQL : INSERT, UPDATE, DELETE, COMMIT, SAVEPOINT, ROLLBACK
Comprend parfois aussi SELECT.
MCD Modèle Conceptuel de données
MLD Modèle Logique de données
MPD Modèle Physique de données
PK Primary Key Clé primaire
NF Normal Form Traduction de "Forme Normale"
1NF = 1st normal form (NF1 en français) etc.
RDBMS Relational Data Base Management System = SGBDR
SGBD Système de Gestion de Base de Données
SGBDR Système de Gestion de Base de Données Relationnelles
