Résumé

an signifie a x a x a ... x a : a multiplié n fois par lui-même.
Exemples :
  • 103 = 10 x 10 x 10 = 1000
  • 25 = 2 x 2 x 2 x 2 x 2 = 32
  • 71 = 7
On pose : a0 = 1, quelque soit a.
On note : a-n = 1 / an
Ex : 10-3 = 1 / 103 = 1 /1000 = 0,001

Formules principales

  • an x am = an + m
    Ex : 103 x 102 = (10 x 10 x 10) x (10 x 10) = (10 x 10 x 10 x 10 x 10) = 105
  • an / am = an - m
    Ex : 24 / 25 = (2 x 2 x 2 x 2) / (2 x 2 x 2 x 2 x 2) = 1 / 2 = 2-1
  • (an)m = an * m
    Ex : (102)3 = 102 x 102 x 102 = 102 + 2 + 2 = 106

Notation scientifique

La notation scientifique d'un nombre est l'écriture de ce nombre de la manière suivante :
  • Un nombre compris entre 1 et 10 (1 ≤ x < 10)
  • Multiplié par une puissance de 10
Ex : 2 859 = 2,859 x 103
Sur une calculatrice (ou dans un langage de programmation), ce nombre est écrit 2.859E3

Exercices

Multiplication
Ecrire sous la forme 10n :
  • 103 x 102 x 104
  • 105 x 10-3
  • 10-2 x 10-5 x 108
  • 104 x 10-4
  • 35 000 x 2 000
  • 33 millions x 1 milliard
  • 0,0008 x 20 000 000
  • 0,002 x 104
Division
Ecrire sous la forme 10n :
  • 103 / 102
  • 105 / 108
  • 102 / 10-2
  • 10-4 / 106
  • 10-4 / 10-6
  • 10-4 / 10-4
  • 18 millions / 60 milliards
Multiplication et division
Ecrire sous la forme 10n :
  • 103 / (10-4 x 102)
  • 105 x 104 / (102 x 10-6)
Ecrire en notation scientifique
(un point signifie "multiplié par")
  • 30 000 x 5 000
  • 180 000 000 / 10-4
  • 3.103 x 2,5.102
  • 2.103 x 5.10-4
  • 6.104 / 3.102
