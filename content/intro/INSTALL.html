INSTALL - Prérequis

La prise en main de l'environnement de travail est donnée en Bash sous Linux (Ubuntu) mais utilisez les moyens que vous voulez. Par exemple vous pouvez travailler sur Powershell ou VSCode ou Eclipse.
L'essentiel est de pouvoir travailler avec git et exécuter des petits programmes dans différents langages.
Note : sudo indique qu'on excute la commande en tant qu'utilisateur root.

git

Installer git
sudo apt install git
Cloner le répertoire du cours
git clone https://github.com/tig12/cnam-2isa.git
Crée un dossier cnam-2isa/ dans le répertoire courant.
Vérifier que vous avez le cours en local en visitant la page cnam-2isa/index.html.
Tester git
cd cnam-2isa/
git remote -v
Pour récupérer en local la dernière version du cours : 
git pull
