INSTALL

Le cours se déroule sous Linux. Une machine virtuelle (VM) est fournie en début de cours.
Sur cette VM, la dernière version LTS de java est installée, ainsi que python, php et git.

Prise en main

La machine virtuelle est fournie sous la forme d'un fichier .ovf (Open Virtualization Format). Pour installer cette VM sur votre machine :
  • Démarrez VMWare
  • Menu File / Open...
  • Sélectionner "Open Virtualization Format VMs (*.ovf *.ova)"
  • Choisissez le fichier ovf fourni pour ce cours et installez la machine
  • Démarrez cette machine virtuelle
Note: pour sortir de la VM et revenir sur votre système habituel, utilisez Ctrl Alt

Sur cette machine : double-click sur "Dossier personnel".
La hiérarchie suivante est disponible : 
/home/toto/
    ├── Bureau
    ├── cours
    │   ├── cnam-2isa   # Copie locale du cours (dépôt git)
    │   └── doc-java    # Copie locale de la doc java
    └── tmp
On est sous Linux (Mint), sous le compte utilisateur toto.
Le mot de passe à utiliser pour effectuer des opérations en mode root est toto
