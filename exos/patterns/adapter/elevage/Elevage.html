Exercice : Elevage de chats

Extrait d'un examen (3 points).

Vous faites partie d'une équipe qui automatise la distribution de nourriture dans un élevage de chats.
Vous avez développé une classe ChatNormal qui contient une méthode alimenter().
Afin de gérer différents types d'alimentation, il a été décidé d'utiliser du code écrit par une société extérieure, Cat Feeders.

Le code existant se trouve dans le dossier enonce/, qui contient :
  • Le code développé par la société Cat Feeders, dans le package com.catfeeders
  • Le code développé par votre équipe, dans le package elevage, qui contient :
    • La classe elevage.animaux.ChatNormal
    • Une classe de test, elevage.Main
Il est impossible de modifier le code de la société Cat Feeders.
Il vous est demandé d'écrire les adaptateurs permettant d'utiliser vos classes de chats et celles de Code Feeders de manière homogène, et de remplacer le code de test existant (dans main()) par du code illustrant cette possibilité : 
        Chat[] chats = {
            /* A compléter */
        };
        
        for(Chat chat : chats){
            chat.alimenter();
        }
