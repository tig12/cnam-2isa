Exercice : Sommateur

Code source de l'énoncé

Compléter Sommateur.java pour obtenir un composite dont :
  • Chaque noeud terminal stocke une valeur de type int.
  • Les noeuds composés ne stockent pas de valeur.
  • Chaque noeud a une méthode somme().
    • Dans un noeud terminal, somme() renvoie sa valeur.
    • Dans un noeud composé, somme() renvoie la somme des valeurs de ses enfants.

Dans la fonction main() :
  • Remplissez le composite avec des valeurs de tests.
  • Affichez la somme totale.
  • Affichez les sommes des enfants directs du noeud racine, dans une boucle "foreach"
