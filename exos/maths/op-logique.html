Les codes suivants sont utilisés pour représenter des couleurs :
  • B : Bleu
  • G : Gris
  • J : Jaune
  • O : Orange
  • N : Noir
  • R : Rouge
  • V : Vert
Complétez les couleurs de la table en appliquant les règles suivantes :
  1. Si A1 est rouge et D2 est vert alors B2 est orange
    Sinon B2 est noire
  2. Si (A1 est rouge et D2 est rouge) ou (C1 est noire et E2 est noire) alors B3 est verte
    Sinon B3 est noire
  3. Si (A1 est rouge ou D2 est rouge) et (C1 est noire ou E2 est noire) alors C3 est jaune
    Sinon C3 est bleue
  4. Si B3 n'est pas verte alors D3 est orange
    Sinon D3 est bleue.
  5. Si D3 est bleue alors B4 est jaune et B5 est grise
    Sinon (
        Si A1 est jaune ou C1 est jaune ou D2 est jaune alors B4 est jaune et B5 est noire
        Sinon B4 est verte et B5 est grise
    )
  6. Si E2 et B4 sont de même couleur alors C3 et D5 sont de même couleur
    Sinon D3 et D5 sont de même couleur
  7. Si la grille contient 1 case rouge et 2 cases noires alors E5 est verte
    Sinon (
        Si D2 est la seule case jaune alors E5 est rouge
        Sinon (
            Si la grille contient plus de cases jaunes que de cases noires alors E5 est bleue
            Sinon (
                Si D3 et O3 sont de même couleur alors E5 est jaune
                Sinon E5 est noire
            )
        )
    )
