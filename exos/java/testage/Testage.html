Exercice : Test de l'âge

Ecrire un programme qui prend en entrée un paramètre (nombre entier).
Si ce nombre est > 18, affichez la chaîne : "La personne est majeure"
Sinon, affichez la chaîne : "La personne est mineure"

Instructions permettant de compiler et exécuter le programme : 
javac TestAge.java
java TestAge 12
java TestAge 34


Aide : pour gérer un paramètre en entrée, on peut utiliser Integer.parseInt(). 
class TestAge{

    public static void main(String[] args){
        // on suppose que la fonction est appelée avec un seul argument
        // et que cet argument représente un nombre entier positif.
        int age = Integer.parseInt(args[0]);
    }
}
Essayez avec différentes valeurs pour age.
Testez en particulier la valeur 18
