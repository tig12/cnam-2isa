Exercice : Voitures

On a le tableau de vente suivant :
2 CV205R5404
Jean0523
Paul4650
Jeanne3043
Paule6207
On donne aussi les prix de chaque voiture :
2 CV205R5404
20 00030 50025 00022 400

Dans une classe Voiture :
  • Créez un tableau à deux dimensions qui contient le tableau des ventes.
  • Créez trois autres tableaux (à une dimension) contenant :
    les noms des vendeurs ou vendeuses ;
    les noms des modèles ;
    les prix des modèles.
  • Ecrivez un programme qui affiche le tableau des ventes avec les totaux :
    - Pour chaque ligne, le chiffre d'affaire du vendeur / de la vendeuse.
    - Pour chaque colonne, le nombre de voitures vendues.
