Exercice : Une première classe

But : écrire une classe, créer des objets de cette classe et les utiliser.
Pour la syntaxe, vous pouvez vous aider de l'exemple de la page sur les classes et les interfaces, ou de la classe située dans Personne.java.

I - Classe Chat

Dans un fichier Chat.java, écrire une classe Chat.
Note : cette classe n'a pas besoin d'avoir une fonction main().

Cette classe a 3 variables d'instance (qui ne sont pas déclarées static) :
  • Le nom, de type String
  • L'âge, de type int
  • La couleur, de type String
Ces 3 variables d'instance doivent être toutes les trois déclarées privées.

La classe Chat a aussi une variable de classe (déclarée static) :
Nombre de pattes, de type int, dont la valeur est 4. Cette variable doit être toutes les trois déclarées privées.
  1. Ecrire un constructeur prenant 3 paramètres : le nom de l'animal, son âge et sa couleur.
  2. Pour chacune des 3 variables d'instance, écrire un getter et un setter.
  3. Ecrire une fonction direBonjour(), qui ne prend aucun paramètre, et qui renvoie une String contenant :
    "Miaou, je m'appelle ", suivi du nom du chat, suivi de son âge, de sa couleur et de son nombre de pattes.
    Par exemple, pour un chat nommé Félix, vieux de 3 ans et de couleur noire, la fonction sePresenter() renvoie : 
    Bonjour, je m'appelle Félix, j'ai 3 ans, ma couleur est le noir, et j'ai 4 pattes.

II - classe TestChat

Dans un fichier TestChat.java, écrire une classe TestChat, qui va contenir une fonction main().
  1. Dans cette fonction main(), créez un chat nommé Minou, qui a 3 ans et qui est gris.
  2. Affichez le résultat de l'exécution de sePresenter() de ce chat.
  3. Modifiez votre chat, en lui affectant un âge de 5 ans.
  4. Affichez de nouveau le résultat de l'exécution de sePresenter() de ce chat. Constatez que cette fonction a bien pris en compte le changement d'âge.
