Exercice : Max et Moyenne

But : manipulations simples sur des tableaux.
Ecrire une classe avec une fonction main() qui initialise un tableau contenant les entiers suivants :
12, 87, 56, 4, -5 et 18

Calculer et afficher :
  • La valeur maximale du tableau
  • L'index contenant cette valeur maximale
  • La moyenne arithmétique des éléments de ce tableau, qui est définie comme la somme des éléments du tableau divisée par le nombre des éléments du tableau.
Le résultat attendu est le suivant : 
java Max
max = 87
index max = 1
moyenne = 28.666666666666668
Indications :
  • Pour calculer la moyenne, pensez à faire du type casting.
