Exercice : ToString

Code source de l'énoncé

Compilez et exécutez la classe ToString.

Implémentez la méthode Personne.toString() en utilisant StringBuilder pour obtenir l'affichage suivant : 
java ToString

AC03 Alonzo Church
GH06 Grace Hopper
AK40 Alan Kay
AT12 Alan Turing
JV03 Johh Von Neumann
La méthode toString renvoie une chaîne composée des initiales de la personne, suivies des deux derniers digits de sa date de naissance, suivis de son nom et de son prénom.
