Exercice : Longueur des éléments

  • Ecrivez une classe avec une fonction main().
  • Dans la fonction main(), créez un tableau de String contenant les éléments suivants : 
    Lisp
Scheme
OCaml
Erlang
Haskell
  • Créez un tableau de int de même longueur que le tableau de String.
  • Dans une boucle, remplissez le tableau de int avec les les longueurs des éléments du tableau de String.
  • Dans une boucle, affichez le résultat : 
    Lisp : 4
Scheme : 6
OCaml : 5
Erlang : 6
Haskell : 7
  • Obtenez exactement même résultat en utilisant l'API stream, sans utiliser de boucle.
